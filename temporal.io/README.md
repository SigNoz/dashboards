# Temporal.io Dashboards

Monitoring dashboards for Temporal, covering Temporal Cloud metrics and the Go and TypeScript SDK metrics.

## Dashboards

- **Temporal Cloud Metrics:** Cloud cluster infrastructure and service-level metrics. [Setup docs](https://signoz.io/docs/integrations/temporal-cloud-metrics/)
- **Temporal SDK Metrics (Go):** Client and worker metrics for Go SDK applications. [Setup docs](https://signoz.io/docs/integrations/temporal-golang-opentelemetry/)
- **Temporal SDK Metrics (TypeScript):** Client and worker metrics for TypeScript/Node.js SDK applications. [Setup docs](https://signoz.io/docs/integrations/temporal-typescript-opentelemetry/)
