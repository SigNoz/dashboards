# ASP.NET Core Metrics Dashboard

Monitors ASP.NET Core applications instrumented with OpenTelemetry, covering HTTP request throughput, latency (P99/P95/P50), error rates, Kestrel memory pool usage, .NET runtime GC behavior, thread pool activity, and routing match outcomes.

For more information, please visit the [Send Metrics from ASP.NET Core using OpenTelemetry doc](https://signoz.io/docs/metrics-management/send-metrics/applications/opentelemetry-aspnetcore/).
