# Nomad Dashboard

This directory contains SigNoz dashboard template for Nomad monitoring.

Browse all dashboard templates in the [SigNoz docs](https://signoz.io/docs/dashboards/dashboard-templates/overview/#available-dashboard-templates).

## Nomad Metrics

**File:** `nomad-metrics-by-host.json`

**Tags:** `nomad`

**Panels:**

- Memory utilization
- Count of objects in heap
- Goroutines
- Raft Last Contact
- Raft Transactions
- Raft Dispatch Log
- Memberlist Gossip
- Raft Apply Count
- Raft Commit Time
- Raft Append Entries
- Raft FSM apply
- Raft Commit Time
- RPC Requests
- RPC Queries
- RPC Errors
- Evaluations
- Server token TTL
- Token Last Renew
- Token Next Renew
- Tokens
