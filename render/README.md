# Render Dashboard

This directory contains SigNoz dashboard template for Render monitoring.

Browse all dashboard templates in the [SigNoz docs](https://signoz.io/docs/dashboards/dashboard-templates/overview/#available-dashboard-templates).

## Render Test

**File:** `render-dashboard.json`

**Filter variables:** `service.instance.id`, `Service_name`

**Sections:** CPU, Memory, Network

**Panels:**

- CPU Time
- CPU Limit
- Memory Usage
- Memory utilization %
- Network in/out
- Memory breakdown
