# Istio Monitoring Dashboard

Monitors Istio service mesh deployments, covering HTTP/TCP traffic throughput, request rates, P50/P95/P99 latency percentiles, error rates, circuit breaker state, mTLS usage, certificate expiry, Envoy data plane connections, and Istiod control plane health (XDS pushes, push latency, proxy convergence time, configuration conflicts).

For more information, please visit the [Istio Metrics doc](https://signoz.io/docs/metrics-management/istio-metrics/).
