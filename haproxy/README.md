# HAProxy Dashboard

This directory contains SigNoz dashboard template for HAProxy monitoring.

Browse all dashboard templates in the [SigNoz docs](https://signoz.io/docs/dashboards/dashboard-templates/overview/#available-dashboard-templates).

## HAProxy dashboard

**File:** `haproxy.json`

**Panels:**

- Network In
- Network out 
- Connection errors
- Connection rate
- Connection retries
- Denied request
- Error requests
- Queued requests
- Requests rate
- Redispatched requests
- Total requests
- Denied responses
- Error responses
- Average sessions
- Sessions count 
- Sessions rate
- 500 errors
