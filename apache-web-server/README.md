# Apache Web Server Dashboard

This directory contains SigNoz dashboard template for Apache Web Server monitoring.

Browse all dashboard templates in the [SigNoz docs](https://signoz.io/docs/dashboards/dashboard-templates/overview/#available-dashboard-templates).

## Apache Monitoring Dashboard

**File:** `apache.json`

**Filter variables:** `apache.server.name`

**Panels:**

- Apache CPU Load
- Average Server Load
- No of Requests
- Traffic
- Apache CPU Time
- Concurrency Efficiency: Average Workers vs. Requests
- Scoreboard
- CPULoad
- Busy workers
- Idle workers
