# PlanetScale Dashboards

These dashboards provide detailed insights into your PlanetScale database branches, monitoring key metrics for query performance, connections, replication, storage, and pod resources.

- **PlanetScale Vitess (MySQL):** vtgate and vttablet query rate, latency, and rows; connection, transaction, stream, and found-rows pools; MySQL internals; replication lag; VTOrc; storage; and pod resources.
- **PlanetScale PostgreSQL:** connections and session states, PgBouncer pooling, commit throughput, locks, WAL archiving, storage, and pod resources.

For setup instructions and more details, please visit the [SigNoz PlanetScale Integration Documentation](https://signoz.io/docs/metrics-management/opentelemetry-planetscale/).
