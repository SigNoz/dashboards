# Memcached Dashboard

Dashboard to monitor Memcached

Browse all dashboard templates in the [SigNoz docs](https://signoz.io/docs/dashboards/dashboard-templates/overview/#available-dashboard-templates).

## Memcached

**File:** `memcached.json`

**Panels:**

- Memcache Server Size (History)
- Memcache Server (Current Size)
- Commands
- Current Connections
- Threads Count
- Total & Current Connections 
- Current Items
- Evictions
