# APM Dashboards

These dashboards monitor service performance from span metrics SigNoz generates from your traces. `apm-metrics` covers RED metrics with database and external-call breakdowns, `db-calls-monitoring` covers database call rate, errors, and top statements, and `http-api-monitoring` covers HTTP endpoint latency, throughput, and status codes.

For setup instructions and metric details, please visit the [SigNoz APM Metrics, Dashboards & Alerts documentation](https://signoz.io/docs/userguide/custom-apm-dashboards-alerts/).
