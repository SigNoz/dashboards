# NVIDIA DCGM Dashboard

This directory contains SigNoz dashboard template for NVIDIA DCGM monitoring.

Browse all dashboard templates in the [SigNoz docs](https://signoz.io/docs/dashboards/dashboard-templates/overview/#available-dashboard-templates).

## NVIDIA DCGM

**File:** `nvidia-dcgm.json`

**Filter variables:** `k8s.cluster.name`, `k8s.node.name`, `GPU`

**Panels:**

- GPU Temperature
- GPU Avg. Temp
- GPU Power Usage
- GPU Power Total
- GPU SM Clocks
- GPU Utilization
- GPU Framebuffer Mem Used
- Tensor Core Utilization
