# MongoDB Dashboard

This dashboard provides a high-level overview of your MongoDB. It includes read/write performance, most-used replicas, collection metrics etc...

Browse all dashboard templates in the [SigNoz docs](https://signoz.io/docs/dashboards/dashboard-templates/overview/#available-dashboard-templates).

## Mongo overview

**File:** `mongodb.json`

**Tags:** `mongo`, `database`

**Filter variables:** `host.name`

**Panels:**

- Operations count
- Total operations time
- Cache operations
- Read latency
- Write latency
- Command latency
- Network IO
- Cursor count
- DB Overview
- Global lock time
