# CouchDB Dashboard

Basic metrics to monitor around CouchDB

Browse all dashboard templates in the [SigNoz docs](https://signoz.io/docs/dashboards/dashboard-templates/overview/#available-dashboard-templates).

## CouchDB

**File:** `couchdb.json`

**Panels:**

- Operations (Reads/Writes)
- Open Databases
- Average Request Time (in ms)
- HTTP Requests
- Success HTTP Responses
- Error HTTP Responses
- Opened File Descriptors
