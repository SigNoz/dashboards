# JMX (Cassandra) Dashboard

This directory contains SigNoz dashboard template for JMX (Cassandra) monitoring.

Browse all dashboard templates in the [SigNoz docs](https://signoz.io/docs/dashboards/dashboard-templates/overview/#available-dashboard-templates).

## Cassandra metrics

**File:** `cassandra.json`

**Filter variables:** `host.name`

**Panels:**

- Token range read request latency
- Standard read request latency
- Standard write request latency
- Total requests by operation
- Error requests by operation and status
- Compactions
- Data size
- Hint messages count
