# CI/CD Dashboard

Dashboard displaying metrics from CI and CD systems.

Browse all dashboard templates in the [SigNoz docs](https://signoz.io/docs/dashboards/dashboard-templates/overview/#available-dashboard-templates).

## CI/CD Observability

**File:** `cicd.json`

**Tags:** `vcs`, `cicd`

**Filter variables:** `repository.name`

**Panels:**

- # Branches (avg)
- # Changes (avg)
- Branch Age (avg)
- Change Approval Time (avg)
- Change Merge Time (avg)
- # Contributors
- Change Age (avg)
- # Changes
- # Changes (detailed)
- # Lines changed
- Change Merge Time (detailed)
- Pipeline Duration (avg)
- Pipeline Duration (P50)
- # Statuses
- # Statuses
- Pipeline Duration (detailed)
